# Skill: Deploy App

## Objective
Lancer localement l’application générée.

## Instructions
1. Inspecter `app_build/` pour identifier la stack.
2. Installer les dépendances appropriées.
3. Lancer l’application avec la bonne commande.
4. Donner à l’utilisateur :
- la commande utilisée
- le port ou l’URL locale
- les éventuels prérequis
