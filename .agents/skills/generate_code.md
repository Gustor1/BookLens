# Skill: Generate Code

## Objective
Générer le code du projet à partir de la spécification approuvée.

## Rules
- Lire `production_artifacts/Technical_Specification.md`.
- Générer les fichiers dans `app_build/`.
- Inclure tous les fichiers nécessaires au démarrage.

## Instructions
1. Lire la spécification validée.
2. Créer l’arborescence du projet.
3. Générer le code principal.
4. Créer les fichiers de dépendances (`requirements.txt`, `package.json`, etc.) si nécessaires.
5. Ne rien inventer qui contredit la spécification.
