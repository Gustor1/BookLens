# Skill: Audit Code

## Objective
Auditer et corriger le code généré.

## Rules
- Comparer le code avec `production_artifacts/Technical_Specification.md`.
- Corriger les bugs, oublis de dépendances, incohérences de structure.
- Réécrire les fichiers corrigés directement dans `app_build/`.

## Instructions
1. Lire la spécification.
2. Inspecter le code de `app_build/`.
3. Vérifier :
- cohérence avec la spec
- imports
- dépendances
- logique générale
- clarté des noms de fichiers
4. Corriger les problèmes trouvés.
